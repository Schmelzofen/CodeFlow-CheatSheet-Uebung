Cheat-Sheet für Terminal

Befehl Effekt
touch "..." erstellt eine neue Datei
cd wechselt den Ordner
mkdir "..." erstellt einen Ordner
pwd zeigt komplette Struktur bis zum Root an
cat "..." öffnet Datei
ls listet Dateien/Ordner im Verzeichnis auf
ls -a listet alle sowie verstecke Dateien auf
cp kopiert/verschiebt/benennt Datei um
rm / rmdir löscht Datei bzw. Ordner

Cheat-Sheet für git

Befehl Effekt
git init initialisiert ein git-Repo
git add . fügt alle Dateien des aktuellen Ordners zur Repo hinzu
git commit -m "..." neue Änderungen hinzufügen
git pull änderungen herunterladen
git push hinzugefügte änderungen hochladen
git branch "..." erstellt eine Abzweigung vom Projekt
git checkout "..." wechselt zum jeweiligen Branch/Zweig
git merge "..." verschmilzt zwei Zweige miteinander
git clone "..." klont das angegebene Repo zur eigenen bearbeitung
git diff zeigt alle vorgenommenen Änderungen an

Erste Schritte zur Erstellung und Veröffentlichung des eigenen Repo's

git init

git add .

git commit -m "..."

git push

---> initialisiert das Repository im lokalen Speicher

---> fügt im aktuellen Ordner alle Dateien zur Staging Zone hinzu

---> speichert änderungen in der Staging Zone

---> lädt Dateien aus der lokalen Staging Zone hoch auf Github

Erstellung sowie Verschmelzung von Branches/Zweigen

git branch "..."

git branch -d "..."

git checkout "..."

git merge "..."

---> erstellt ein Branch mit dem Namen "..."

---> löscht die Branch mit folgendem Namen "..."

---> wechselt zur Branch mit dem Namen "..."

---> verschmiltzt die Branch und übernimmt alle Daten von folgender Branch "..."
