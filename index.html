Cheat-Sheet für Terminal

Befehl Effekt
touch "..." erstellt eine neue Datei
cd wechselt den Ordner
mkdir "..." erstellt einen Ordner
pwd zeigt komplette Struktur bis zum Root an
cat "..." öffnet Datei
ls listet Dateien/Ordner im Verzeichnis auf
ls -a listet alle sowie verstecke Dateien auf
cp kopiert/verschiebt/benennt Datei um
rm / rmdir löscht Datei bzw. Ordner

Cheat-Sheet für git

Befehl Effekt
git init initialisiert ein git-Repo
git add . fügt alle Dateien des aktuellen Ordners zur Repo hinzu
git commit -m "..." neue Änderungen hinzufügen
git pull änderungen herunterladen
git push hinzugefügte änderungen hochladen
